# Online Bookstore Management System

## Overview

The Online Bookstore Management System is a web-based application developed to simulate an interactive online bookstore experience. The project allows users to browse books, view pricing information, and complete a simple payment workflow through a clean and user-friendly interface.

This project was developed to strengthen practical skills in frontend development, software structure, debugging, and user interaction design.

---

## Features

* Interactive bookstore homepage
* Book listings with pricing information
* Payment page simulation
* Successful payment confirmation page
* Responsive and user-friendly interface
* Navigation between multiple pages
* Dynamic functionality using JavaScript

---
